# 💱 Currency Converter

A simple and responsive Currency Converter built using HTML, CSS, and JavaScript. It allows users to convert currencies in real-time using exchange rate APIs.

## 🚀 Features

- Real-time currency conversion
- Multiple currency support
- Country flags display
- Responsive design
- User-friendly interface

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
